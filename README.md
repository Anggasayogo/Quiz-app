# Quiz App Using React Js :-)
Quiz app adalah Aplikasi quiz online dimana kamu bisa menjawab semua pertanyaan pilihan ganda yang disediakan !

## instalasi
```
git clone https://github.com/Anggasayogo/Quiz-app.git
```
```
npm install
```
```
npm run start
```
